Feature Status Notes
Properties
.separator Ready The path separator for O/S.
.workingDirectory Ready The current working directory.
Query Methods
.list(path) Ready Returns list of files in path.
.absolute(path) Ready Returns absolute path to file or directory
.exists(path) Ready true if a file or a directory exists.
.isDirectory(path) Ready true if specified path is directory.
.isFile(path) Ready true if specified path is a file
.isAbsolute(path) Ready true for an absolute path
.isExecutable(path) Ready true if file can be executed
.isReadable(path) Ready true if file or directory is readable
.isWritable(path) Ready true if file or directory is writable
.isLink(path) Ready true if path is symbolic link (not junction)
.readLink(path) Ready returns the target of a symbolic link
Directory Methods
.changeWorkingDirectory(path) Ready Changes the current .workingDirectory
.makeDirectory(path) Ready Creates a new directory.
.makeTree(path) Ready Creates a directory including missing parent directories.
.removeDirectory(path) Ready Removes a directory if it is empty.
.removeTree(path) Ready Removes specified path and everything in it.
.copyTree(source, destination) Ready Copies all files from source to destination path.
File Methods
.open(path, mode) Ready Returns a stream object (mode: 'r' 'w' 'a' or ‘rb’ ‘wb’);
.read(path) Ready Reads contents of a file.
.write(path) Ready Writes content to a file (mode: 'w' 'a' or 'wb').
.size(path) Ready Returns size (in bytes) of a file.
.remove(path) Ready Removes a file in specified path
.copy(source, destination) Ready Copies a file to destination path.
.move(source, destination) Ready Moves a file to another, effectively renaming it.
.touch(path) Ready Touches a file, changing its access timestamp.

Stream Object

A stream object returned from the fs.open(path) function has the following methods:

FEATURE STATUS NOTES
Methods
.read() Ready Returns the content of the stream.
.readLine() Ready Reads a line from the stream and returns it.
.atEnd() Ready Returns true if the end of the file was reached.
.write(data) Ready Writes a data string to the stream.
.writeLine(data) Ready Writes the data string as a line to the stream.
.seek(pos) Ready Moves to a certain position into a stream.
.flush() Ready Flushes all pending input/output.
.close() Ready Closes and completes the stream operation.
